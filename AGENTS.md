# AGENTS.md

## Project overview

This is a personal website for Isaiah Peebles.

Purpose:

- Showcase skills and projects
- Present a professional personal brand
- Make it easy for recruiters or collaborators to understand value quickly

## Audience

- Hiring managers
- Recruiters
- Engineers / technical teams

## Goals

- Clear, direct communication
- Clean and modern UI
- Fast loading and simple structure
- Mobile-friendly
- No unnecessary complexity

## Pages

- index.html → landing + value proposition
- about.html → background and skills
- projects.html → portfolio and work examples
- contact.html → contact methods

## Content rules

- Keep writing concise and direct
- Avoid fluff or vague statements
- Focus on outcomes and impact
- Prefer bullet points over long paragraphs

## Design rules

- Minimalist design
- Consistent spacing and typography
- Strong visual hierarchy (headings matter)
- Avoid clutter

## Coding rules

- Use semantic HTML
- Keep CSS simple and reusable
- Use vanilla JavaScript only
- Do not introduce frameworks

## Project rules

- Do not rename files unless required
- Do not restructure folders unnecessarily
- Reuse components/styles when possible

## Resume

- Store resume in /assets/resume.pdf
- Ensure it is linked clearly on homepage

## Validation checklist

Before finishing:

1. Site loads without errors
2. Layout works on mobile
3. Navigation links work
4. No console errors
5. Summarize changes made
