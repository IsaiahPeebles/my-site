Featured Project

Excel Change Log PDF Tool

Problem

Reviewing changes across Excel-based documentation was time-consuming and difficult to communicate clearly, especially when revisions needed to be shared in a format others could quickly understand.

What I built

Developed a Python-based workflow that compares Excel files, identifies changes across sheets, and generates a structured PDF change log for easier review and communication.

Result / Impact

Reduced manual comparison effort, improved revision visibility, and made document changes easier for stakeholders to review and interpret.

Tools Used

Python, openpyxl, PyMuPDF, Excel-based workflow automation
